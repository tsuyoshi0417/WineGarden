# 🌿 WineGarden

MacでWindowsのエロゲを快適に遊ぶためのランチャーアプリ。

## 必要なもの

- macOS 13 (Ventura) 以降
- [Wine Stable](https://www.winehq.org/) （Applicationsフォルダに入れるだけ）

## 使い方

1. WineGarden.appを起動
2. 「＋ 追加」でゲームの.exeファイルを選択
3. 「ゲームを起動」ボタンを押すだけ

設定（Retina対応・日本語ロケール・WINEPREFIXなど）は全て自動で行われます。

## 対応エンジン

- NScripter2 (NS2)
- ※ その他のエンジンは順次対応予定
